# Inbox Processor

Process items in 00_Inbox using the GTD method.

## The 2-Minute Rule

If it takes less than 2 minutes, do it now:
- Reply to simple messages
- File obvious items
- Delete obvious junk

## The Decision Tree

For each item:

1. **Trash** - Delete it
2. **Reference** - Move to 03_Resources
3. **Task** - Move to 01_Projects or Area
4. **Someday/Maybe** - Move to 04_Archive
5. **Next Action** - Create task in project

## Processing Order

1. Oldest first
2. One item at a time
3. Decide, don't deliberate
4. When in doubt, reference it

## Output

```
Inbox Summary:
- Processed: X items
- Trashed: X
- Filed to Projects: X
- Filed to Resources: X
- Remaining: X
```
