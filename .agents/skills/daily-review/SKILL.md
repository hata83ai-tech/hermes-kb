# Daily Review

Reflect on today's work and prepare for tomorrow.

## Review Structure

### 1. What Did We Accomplish?
List concrete outputs - tasks completed, decisions made, problems solved.

### 2. What Didn't Get Done?
Acknowledge without judgment - what was planned vs actual.

### 3. What Did We Learn?
Capture insights - new information, surprises, mistakes to avoid.

### 4. Tomorrow's Priority
Top 3 items for tomorrow.

## Output Format

```
## Daily Review - [Date]

### Accomplishments
- ...

### Incomplete
- ...

### Learnings
- ...

### Tomorrow
1. ...
2. ...
3. ...
```

## Pro Tips

- Do this at the same time daily for habit
- Keep it brief (5-10 minutes max)
- Celebrate wins, however small
